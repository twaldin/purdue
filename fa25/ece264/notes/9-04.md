**class**
---

- "type * ptr = "
    create ptr with type
- "* p ="
    assign a value to the value to the address that p points to
- "= * p"
    read the value at the address that p points to

use & to derefrence normal variable. eg:

int a = 9;
int * p = &a;
now print("%d", * p) will print 9.
